Detailed Description

represents a 'trial' of the coin tossing scenario

a coin is tossed to choose between two other coins and then the chosen coin is tossed some number of times

eg. the show method will make something like the following be displayed CHOICE: A TOSSES: HHHHHHHHTT H:8 T:2
